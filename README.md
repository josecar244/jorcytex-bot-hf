# Jorcytex Bot - Hugging Face Version

Este es el bot de atención al cliente para **Inversiones JORCYTEX EIRL**, optimizado para correr en Hugging Face Spaces.

## 🚀 Configuración
Asegúrate de configurar los siguientes **Secrets** en Hugging Face:
- `GROQ_API_KEY`
- `GOOGLE_API_KEY`
- `PINECONE_API_KEY`
- `SUPABASE_URL`
- `SUPABASE_KEY`
- `LANGFUSE_PUBLIC_KEY`
- `LANGFUSE_SECRET_KEY`
- `LANGFUSE_BASE_URL`
- `EVOLUTION_API_URL`
- `EVOLUTION_API_KEY`
- `EVOLUTION_INSTANCE`
